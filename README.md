# Pathfinding Visualizer

## Overview
This project is an interactive grid-based pathfinding simulator developed using Python and Tkinter.  
It demonstrates classical Artificial Intelligence search algorithms with real-time visualization.

## Implemented Algorithms
- A* Search Algorithm
- Greedy Best First Search (GBFS)

## Heuristic Functions
- Manhattan Distance
- Euclidean Distance
- Chebyshev Distance

## Features
- Adjustable grid size
- Interactive obstacle placement
- Diagonal movement option
- Dynamic path visualization
- Real-time exploration and path highlighting

## Requirements
Python 3.10 or higher  
No external libraries required

## How to Run

python path_visualizer.py
